# 見やすいWebページを作ろう — Gemma があなたのデザインを分析する体験型ゲーム

千葉工業大学 認知情報科学科のオープンキャンパス体験用として開発された、Webデザインの基本原則をインタラクティブに学ぶ教育型ゲームです。

**[Vercel デプロイ済サイトはこちら](https://web-design-evaluator-gemma.vercel.app/)**

## ✨ 一言紹介 / こだわりポイント
千葉工業大学認知情報科学科のオープンキャンパスにおける学び体験コーナーで使えるような、Webページを動的にデザインするゲームを作成した。評価にはGoogle Gemini APIを用いている。わかりやすいUIと学科の学習内容に関連させることを心掛けた。

## 🎮 プログラムの説明（ゲームの目的）

このゲームは、**「なぜ読みやすいデザインが良いのか」を体験的に理解すること**を目的としています。
Webデザインの善し悪しは、テキストで説明されても実感しにくい抽象的な概念です。そこで、「自分でパラメータを動かす → リアルタイムで変化を確認する → AIに採点してもらう」というゲームサイクルを通じて、視認性や認知負荷といったデザインの物理法則（認知科学）を直感的に学べるように設計しました。

## 📱 ページの説明とゲームの流れ

アプリケーションは SPA（Single Page Application）として構築されており、シームレスに以下の画面を遷移します。

1. **タイトル画面**: ゲームの入り口。APIの接続状態もここで確認します。
2. **遊び方説明画面**: 簡単なルールと評価基準（視認性・構成・認知負荷）を説明します。
3. **ゲーム画面**: 左側のパネルで14種類のデザインパラメータ（色、文字サイズ、レイアウトなど）を操作し、右側でリアルタイムにプレビュー結果を確認しながらWebページをデザインします。
4. **評価画面**: 完成したデザインをGoogleのAI「Gemma」が3つの観点から分析し、スコア、グレード（S〜D）、レーダーチャート、具体的な改善アドバイスを提示します。
5. **解説画面**: デザインの背後にある「認知科学的アプローチ」や、評価に使用している「LLM（大規模言語モデル）の仕組み」を学べる、1ページ完結のリッチな解説コンテンツです。

## 🏗️ 構造（技術スタックと構成）

本プロジェクトは **Antigravity + GitHub + Vercel** の環境を活用して開発・デプロイされています。

### フロントエンド
- **HTML5 / CSS3**: グラスモーフィズムを取り入れたモダンで直感的なUI。
- **JavaScript (Vanilla)**: フレームワークに依存せず、DOM操作と非同期通信を用いたSPAを実現。
- **Chart.js**: 評価結果のレーダーチャートを動的に描画。

### バックエンド
- **Python 3 / FastAPI**: 軽量かつ高速な非同期Webフレームワーク。
- **Google GenAI SDK**: GoogleのGemini API（Gemmaモデル）との通信を担当。
- 複数のGemmaモデルを優先順位順に呼び出す**フォールバック機構**を実装し、安定したAPI通信を確保。

### 評価プロンプトの工夫
バックエンドでは、フロントエンドから受け取ったデザインパラメータ（数値や色名）を元に、3つの独立したプロンプト（視認性、構成、認知負荷）を動的に生成します。AIに「プロのWebコンサルタント」というペルソナを与え、JSON形式で安定した評価を返させる高度なプロンプトエンジニアリングを実装しています。

## 💡 作成理由

千葉工業大学での学び（認知情報科学科での知見や、Web3/AI概論での学び）を形にするためです。単なる便利ツールではなく、「作って楽しい」「評価されて悔しい・嬉しい」「解説を読んで納得する」という一連の体験を通じて、オープンキャンパスに来た高校生や一般の方に、デザインとAIの面白さを同時に感じてもらえるプロダクトを目指しました。
